# Install on nakama server
```
~/servers/nakama> git clone https://github.com/talktonpc/nakama-modules.git {*folder* v0.2}
```

# Run with docker-compose
```
~/servers/nakama> cd {*folder* v0.2}
~/servers/nakama/v0.2> docker-compose up -d
```
